                          HOW TO USE
================================================================================

1. READ THE DOCUMENTATION
   → Start with README.md in project root
   → Or GETTING_STARTED.md for quick setup

2. INSTALL THE LIBRARY
   → Follow GETTING_STARTED.md steps
   → Add dependencies to build.gradle

3. INITIALIZE IN APPLICATION CLASS
   → Create AdsConfig
   → Call AdsManager.initialize()

4. USE IN ACTIVITIES
   → Use AdHelper methods
   → Or AdsManager for more control

5. HANDLE CALLBACKS
   → Set AdListener for callbacks
   → Or collect() events from Flow

6. CLEAN UP
   → Call destroy() in onDestroy()
   → Call AdHelper.destroyAds()

================================================================================
                       QUICK START (5 MINUTES)
================================================================================

Step 1: Add Dependency
   implementation("com.github.theankitparmar:adsmanager:2.0.0")

Step 2: Initialize
   val config = AdsConfig(
       isTestMode = BuildConfig.DEBUG,
       bannerAdUnitId = "YOUR_AD_UNIT_ID"
   )
   AdsManager.initialize(this, config)

Step 3: Show Banner Ad
   AdHelper.showBannerAd(
       context = this,
       container = findViewById(R.id.banner_container)
   )

Step 4: Clean Up
   override fun onDestroy() {
       super.onDestroy()
       AdHelper.destroyAds()
   }
